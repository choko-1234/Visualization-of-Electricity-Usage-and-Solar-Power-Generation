# 九州電力の電力需給実績可視化
## プロジェクトの概要
九州エリアにおける電力使用実勢と太陽光発電量を比較することで、実際の電力使用量に太陽光発電がどの程度貢献できているかを可視化できるようにする。また電力使用量の推移から、需要が増す時間帯を特定し将来的な需要の予測に貢献する。

## 考察
・日中(12時~15時)の電力使用量に対し、太陽光発電量は使用量の **約66%** をカバーできている。  
・夕方18時から19時にかけて電力使用量は最も多くなるが、太陽光発電量がカバーできているのは0%である。そのためこの時刻における代替電力が必要になると考える。  
・19時以降電力使用量が右肩下がりなのは家事の終了、商業施設の閉店、段階的な就寝によるものだと推測できる。

## 技術的な工夫
・**データ整形**::日付を示す文字列を日付型に変換し、さらに時刻と合体させることで複数日での比較やグラフの拡張性を持たせた。  
・**データクレンジング**:九州電力のcsvファイルは文字コードがShift-jJISであったりヘッダーが途中にあったため、pandasのskiprowsやencodingオプションを用いて処理を行った
